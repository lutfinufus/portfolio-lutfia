Data Cleaning

This process aims to identify and correct inconsistent data, such as duplicates, missing values, and other discrepancies. It ensures that the data is accurate and consistent, thereby facilitating easier analysis and reporting.

Identify Duplicates

Duplicate values in each column are generally not allowed in data tables but can be present in new, raw, or uncontrolled data. To verify whether a column contains duplicates, the following SQL command can be used:

  • COUNT: An aggregate function that counts all rows or specific rows, identifying records that match certain conditions.
  • GROUP BY: A clause used to group all rows based on the same column value. This clause is typically used with aggregate functions such as COUNT, MAX, MIN, SUM, AVG, etc. The HAVING clause is used with GROUP BY to exclude groups that do not fulfill certain criteria.
  • WHERE: This clause is used to specify criteria or conditions for retrieving data from a table. When combined with the AND logical operator, it ensures that all specified criteria are fulfilled for the query to execute.
